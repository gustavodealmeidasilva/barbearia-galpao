# Barbearia Galpão

Landing page fictícia para uma barbearia tradicional gaúcha, desenvolvida como projeto de portfólio com HTML, CSS e JavaScript puros.

A proposta do projeto é apresentar uma barbearia com identidade forte, visual rústico/premium e linguagem regional. A página combina tons escuros, dourado, imagens de barbearia, tipografia editorial e chamadas diretas para agendamento.

## Demo

Acesse o projeto publicado:

[https://gustavodealmeidasilva.github.io/barbearia-galpao/](https://gustavodealmeidasilva.github.io/barbearia-galpao/)

## Objetivo do projeto

Este projeto foi criado para praticar a construção de uma landing page comercial completa, simulando uma página real para captação de clientes de uma barbearia.

O foco principal foi desenvolver:

- Estrutura HTML semântica
- Layout responsivo
- Identidade visual coerente com o nicho
- Seções comerciais bem organizadas
- Chamada clara para agendamento
- Experiência simples em dispositivos móveis

## Funcionalidades

- Navegação fixa no topo da página
- Menu hambúrguer para telas menores
- Hero com imagem de fundo e botões de ação
- Seção de diferenciais com números de destaque
- Lista de serviços com descrição e preço
- Seção sobre a história da barbearia
- Galeria de imagens com efeito de zoom
- Depoimentos de clientes
- Área de contato com endereço, horários e WhatsApp
- Rolagem suave entre seções
- Layout adaptado para desktop, tablet e celular

## Seções da página

### Hero

Apresenta a Barbearia Galpão com uma chamada forte, subtítulo de impacto e botões para agendamento e visualização dos serviços.

### Diferenciais

Mostra dados rápidos sobre a barbearia, como anos de experiência, clientes atendidos e barbeiros especializados.

### Serviços

Lista os principais serviços oferecidos, incluindo corte clássico, barba na navalha, degradê, hidratação e pacote completo.

### Sobre

Conta uma história fictícia da marca, reforçando tradição, atendimento artesanal e conexão regional.

### Galeria

Exibe imagens relacionadas ao universo da barbearia, com efeito visual de zoom e filtro.

### Depoimentos

Inclui comentários fictícios de clientes para reforçar confiança e prova social.

### Contato

Reúne endereço, telefone, Instagram, horários de funcionamento e chamada para envio de mensagem pelo WhatsApp.

## Como rodar localmente

1. Clone o repositório:

```bash
git clone https://github.com/gustavodealmeidasilva/barbearia-galpao.git
```

2. Acesse a pasta do projeto:

```bash
cd barbearia-galpao
```

3. Abra o arquivo `index.html` no navegador.

Também é possível usar uma extensão como Live Server no VS Code.

## Como publicar

Este projeto pode ser publicado facilmente em:

- GitHub Pages
- Netlify
- Vercel

Como é um projeto estático, não precisa de backend, banco de dados ou processo de build.

## Personalização

Para adaptar o projeto para uma barbearia real, altere:

- Nome da marca
- Endereço e cidade
- Número de WhatsApp
- Link do Instagram
- Serviços e preços
- Depoimentos
- Imagens da galeria
- Cores e fontes no CSS
