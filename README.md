# Barbearia Galpão

Landing page fictícia para uma barbearia tradicional gaúcha, criada como projeto de portfólio com HTML, CSS e JavaScript puros.

O projeto trabalha uma identidade visual rústica e premium, com tons escuros, dourado, tipografia editorial e linguagem regional para transmitir tradição, confiança e atendimento artesanal.

## Preview

Abra o arquivo `index.html` no navegador para visualizar a página localmente.

## Funcionalidades

- Layout responsivo para desktop, tablet e celular
- Navegação fixa com links para as seções da página
- Menu hambúrguer no mobile
- Hero com imagem de fundo, chamada principal e botões de ação
- Bloco de diferenciais com números de destaque
- Grade de serviços com preços
- Seção sobre a história da barbearia
- Galeria com imagens externas e efeito de zoom
- Depoimentos de clientes
- Área de contato com endereço, horários e chamada para WhatsApp
- Rolagem suave entre seções

## Seções da página

- **Hero:** apresentação da Barbearia Galpão e chamadas para agendamento.
- **Diferenciais:** anos de experiência, clientes atendidos e equipe.
- **Serviços:** lista de cortes, barba, pacotes e tratamentos.
- **Sobre:** história fictícia da marca e fundador.
- **Galeria:** imagens de cortes, ambiente e detalhes da barbearia.
- **Depoimentos:** prova social com comentários de clientes.
- **Contato:** informações de localização, horários e botão para WhatsApp.

## Tecnologias

## Como rodar

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Abra o arquivo `index.html` no navegador.

Também é possível usar uma extensão como Live Server no VS Code para visualizar a página com recarregamento automático.

## Personalização

Para adaptar o projeto para uma barbearia real, altere principalmente:

- Nome da marca no `index.html`
- Endereço, telefone, Instagram e link do WhatsApp
- Preços e nomes dos serviços
- Textos da seção sobre e dos depoimentos
- Imagens usadas na hero e na galeria
- Cores e fontes em `css/style.css` ou `css/variables.css`

## Observação

Este projeto foi desenvolvido para fins de estudo e portfólio. As informações comerciais, depoimentos e dados da barbearia são fictícios.
